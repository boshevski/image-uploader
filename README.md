#### Install
```
npm install
```

### Running
Start MongoDB (Windows):
```
C:\Program Files (x86)\MongoDB\Server\3.2\bin>mongo.exe --dbpath <path to dir>
```

Start the app:
```
node app.js
```

### Testing
```
mocha
```

### Linting
```
npm run linter
```
